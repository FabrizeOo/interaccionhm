MODO 1: ALTO CONTRASTE
Para personas con baja visión (deficiencia visual severa pero no ceguera)
Paleta de color:

Fondo página: #FFFFFF (blanco puro)
Texto principal: #000000 (negro puro)
Navbar: #000000 (fondo) + #FFFFFF (texto)
Acentos/botones activos: #FFD600 (amarillo)
Bordes de tarjetas: #000000 con grosor 2px
Tarjetas: fondo blanco #FFFFFF con borde negro

Tipografía:

Fuente: Arial o Verdana (sin remates, legible)
Tamaño mínimo texto: 16px
Peso: Regular para body, Bold para encabezados
Interlineado: 1.5 líneas (espaciado generoso entre líneas)
Espaciado entre palabras: normal

Elementos específicos:

Las 5 tarjetas (Exposición, Presidentes, Senadores, Diputados, Parlamento): fondo BLANCO, bordes NEGROS 2px, íconos NEGROS
El toggle de este modo está ACTIVO: círculo amarillo #FFD600 a la derecha, fondo gris oscuro #F0F0F0
Los otros toggles (Alto Contraste Estricto, Daltonismo): inactivos = gris claro #E0E0E0, sin relleno
Botones de navegación: texto negro sobre fondo blanco con borde negro

Contraste verificado:

Negro sobre blanco = 21:1 (supera WCAG AAA)
Amarillo #FFD600 sobre negro = 12:1 (supera WCAG AAA)

Lo que NO debe haber:

Grados de gris (solo blanco/negro/amarillo)
Colores pastel o suaves
Transparencias que debiliten el contraste


MODO 2: ALTO CONTRASTE ESTRICTO
Para personas con baja visión extrema o sensibilidad a luz
DIFERENCIA CLAVE CON MODO 1:

NO usar blanco como fondo (puede causar resplandor)
TODO es negro puro O amarillo puro (cero grises, cero medios tonos)
Texto AMARILLO puro (#FFFF00), NO blanco

Paleta de color:

Fondo página: #000000 (negro puro)
Texto principal: #FFFF00 (amarillo puro)

Tipografía:

Fuente: Arial o Verdana (sin remates)
Tamaño mínimo: 18px (MÁS GRANDE que modo 1)

Interlineado: 1.8 líneas (aún más separado)

Elementos específicos:

Las 5 tarjetas: fondo NEGRO #000000, bordes AMARILLO 3px, texto AMARILLO, íconos AMARILLO
El toggle de este modo está ACTIVO: círculo amarillo #FFFF00 a la derecha, fondo NEGRO
Los otros toggles: inactivos = fondo negro, texto gris #666666
Separadores/líneas: #FFFF00

Contraste verificado:

Amarillo puro sobre negro = 19.56:1 (supera WCAG AAA + es máximo posible)

Lo que NO debe haber:

Ningún gris (ni siquiera para elementos inactivos)
Ningún blanco (causa deslumbramiento)
Colores intermedios
Transparencias


MODO 3: DALTONISMO (Paleta segura deuteranopia/protanopia)
Para personas que NO distinguen rojo-verde (tipo más común de daltonismo)
CONCEPTO CRÍTICO:

Las personas con daltonismo pueden tener dificultades para diferenciar entre el rojo y el verde. Este modo NO es oscuro ni contrasta extremo: es sobre usar COLORES DISTINGUIBLES. Además, los elementos no deben transmitir información únicamente mediante el color — si algo depende del color para ser entendido, añade PATRÓN, BORDE o ÍCONO diferenciador. userwayuserway
